# 营养知识参考指南

> 本文件供 AI 助手在推荐饮食时参考。核心理念：**结构为主，卡路里为辅**。日常推荐看食物结构（蛋白质够不够/蔬菜有没有），不追精确数字。但隐形热量陷阱必须揭示，周报给粗略趋势，用户问了就答。

---

## 一、目标导向的饮食策略

### 减脂
- **核心**：高蛋白 + 多蔬菜 + 控碳水，不是饿肚子
- **每餐结构**：½ 蔬菜 + ¼ 蛋白质 + ¼ 主食（换成粗粮更好）
- **食堂实操**：打菜时多打一个素菜，主食减半或换粗粮
- **避坑**：避开糖醋/红烧/油炸类窗口，选清炒/蒸/煮

### 增肌
- **核心**：足量蛋白质 + 适量碳水 + 力量训练后补餐
- **每餐结构**：⅓ 蛋白质 + ⅓ 碳水 + ⅓ 蔬菜
- **食堂实操**：每餐至少一个肉菜，加一个蛋
- **加餐**：训练后喝牛奶/吃鸡蛋/香蕉

### 省钱
- **核心**：用最便宜的蛋白质来源填饱肚子
- **高性价比蛋白**：鸡蛋 < 豆腐 < 鸡胸肉 < 牛奶 < 鱼肉
- **食堂实操**：选自选窗口而非小炒，一荤两素比两个半荤便宜
- **避坑**：少吃外卖（配送费+包装费占30%），食堂堂食最划算

### 健康饮食（通用/稳定期）
- **核心**：食物多样 + 少油少盐 + 规律三餐
- **每餐至少**：1个蛋白质 + 2种蔬菜 + 1份主食
- **食堂实操**：每周吃够5种以上蔬菜，换着窗口吃

## 二、目标-relative菜品评分

> **核心原则**：评分不是绝对营养评分，而是"这顿对用户当前目标来说合不合理"。

同一顿饭对不同目标评分不同：

| 餐食 | 减脂期 | 增肌期 | 稳定期 |
|------|--------|--------|--------|
| 番茄炒蛋+米饭 | 7分（蛋白够碳水适中） | 5分（蛋白偏少） | 8分（均衡） |
| 红烧肉+米饭 | 4分（太油碳水多） | 7分（蛋白足） | 6分（偶尔可以） |
| 清炒西兰花+白粥 | 5分（低碳水但蛋白几乎没有） | 3分（蛋白严重不足） | 5分（缺蛋白） |
| 白切鸡+时蔬+半份饭 | 8分（低油高蛋白蔬菜足） | 7分（蛋白足碳水偏少） | 8分（均衡） |
| 炸鸡腿+奶茶 | 2分（油炸+高糖饮品） | 5分（蛋白有但油太多） | 4分（偶尔放纵） |

**评分依据**：
- **减脂期**：蛋白质是否够 + 蔬菜是否≥1份 + 碳水是否适量 + 有无隐形陷阱
- **增肌期**：蛋白质是否足量(≥2个来源) + 碳水是否支撑训练 + 蔬菜有无
- **稳定期**：整体均衡度 + 食物多样性 + 有无明显偏科

## 三、隐形热量陷阱速查表

> **检测到陷阱时必须揭示，附带参照物。不追精确数字，但必须让用户意识到"看着不多但热量不少"。**

### 常见陷阱

| 食物 | 看起来 | 实际热量 | 参照物 | 替代方案 |
|------|--------|---------|--------|---------|
| 一小碟花生米(30g) | "零食几口" | ≈170卡 | ≈半份米饭 | 酒精过敏→酸奶≈80卡/苹果≈80卡 |
| 沙拉+酱(1份) | "很健康" | 酱≈200卡 | ≈1份白米饭 | 少放酱/选醋汁 |
| 奶茶/糖水(1杯) | "喝的不多" | ≈300-500卡 | ≈一顿正餐 | 无糖茶/白开水 |
| 油炸鸡排(1份) | "一份肉而已" | ≈500卡 | ≈2份白米饭+蒸鸡胸≈150卡 | 白切鸡/蒸鸡胸 |
| 糖醋排骨(1份) | "排骨有肉" | ≈400卡 | ≈1.5份米饭 | 清炖排骨/红烧少糖版 |
| 炒饭/炒面(1份) | "主食一份" | ≈500-600卡 | ≈2.5份白米饭 | 白米饭/杂粮饭 |
| 油条(1根) | "早餐小吃" | ≈250卡 | ≈1份米饭 | 鸡蛋/包子/馒头 |
| 蛋挞/蛋糕(1个) | "小小甜点" | ≈300卡 | ≈1份米饭 | 水果/酸奶 |
| 奶油蘑菇汤(1碗) | "汤很暖" | ≈250卡 | ≈1份米饭 | 清汤/番茄蛋汤≈100卡 |
| 热干面(1份) | "面条而已" | ≈450卡 | ≈2份米饭 | 清汤面≈200卡 |

### 参照物体系

报卡路里时**永远附带参照物**，不让用户面对裸数字：

| 参照物 | 约等于 |
|--------|--------|
| 半份米饭（100g） | ≈120卡 |
| 一份米饭（200g） | ≈240卡 |
| 一顿正常午餐 | ≈600-800卡 |
| 一顿减脂期午餐 | ≈400-600卡 |
| 减脂期一天总量 | ≈1500-1800卡 |
| 增肌期一天总量 | ≈2200-2800卡 |

## 四、生理期饮食策略（女性用户）

### 经前期（来之前3-5天）
- **策略**：补铁、减少盐分（防水肿）、补充维生素B6（缓解情绪）
- **食堂推荐**：红肉（牛肉/猪肝）、菠菜、香蕉、坚果
- **避免**：过咸食物、过多甜食（会加重水肿和情绪波动）

### 经期中（3-7天）
- **策略**：**忌冰、忌辣**、温热食物、补铁补血
- **食堂推荐**：热汤（番茄蛋汤/排骨汤）、红枣枸杞、温性食物（鸡肉/羊肉）、深色蔬菜
- **绝对避免**：冰饮、凉拌菜、生冷水果（西瓜/梨）、辛辣刺激（辣椒/花椒）
- **饮品替代**：温水/红糖姜茶/热豆浆 代替 冰奶茶/冰咖啡

### 经期后（结束后7天）
- **策略**：补血恢复、逐渐恢复正常饮食
- **食堂推荐**：红枣/枸杞/动物肝脏/红肉/深色蔬菜
- **可恢复**：正常饮食，但仍建议少冰

## 五、四季饮食指南

### 春（3-5月）
- **推荐倾向**：清淡、多吃绿叶蔬菜、少油腻
- **食堂推荐**：清炒时蔬、春笋、荠菜、豌豆
- **注意**：春天容易过敏，注意海鲜类食物

### 夏（6-8月）
- **推荐倾向**：清凉、补水、瓜果、可吃凉拌
- **食堂推荐**：凉拌黄瓜、西瓜、绿豆汤、苦瓜、冬瓜汤
- **注意**：别只吃冰的，冷热搭配；注意食物保鲜

### 秋（9-11月）
- **推荐倾向**：润燥、银耳/梨/蜂蜜
- **食堂推荐**：银耳羹、梨、蜂蜜水、百合、莲藕
- **注意**：少辛辣（秋燥），避免上火

### 冬（12-2月）
- **推荐倾向**：温热、汤类、根茎类
- **食堂推荐**：排骨汤、羊肉汤、红薯、萝卜炖牛腩、热粥
- **绝对避免**：冰饮、糖水、凉拌菜（冬天推这些不合适）
- **注意**：冬季需要更多热量，可适当增加主食分量

## 六、食物分组速查

### 蛋白质（每餐必备）
| 档位 | 食堂常见菜品 | 特点 |
|------|-------------|------|
| 优秀 | 白切鸡、蒸蛋、水煮虾、卤牛肉、豆腐 | 低油高蛋白 |
| 一般 | 番茄炒蛋、宫保鸡丁、鱼香肉丝 | 有蛋白但油大 |
| 避免 | 炸鸡腿、糖醋排骨、红烧肉 | 高油高糖 |

### 蔬菜（每餐至少1-2种）
| 类型 | 食堂常见菜品 | 建议 |
|------|-------------|------|
| 绿叶 | 清炒时蔬、蒜蓉青菜 | 优先选，维生素多 |
| 根茎 | 土豆丝、胡萝卜丝 | 当蔬菜吃，但碳水偏高 |
| 混炒 | 地三鲜、干锅花菜 | 油偏多，适量吃 |

### 主食（控制分量）
| 类型 | 食堂常见 | 建议 |
|------|---------|------|
| 精制 | 白米饭、白面条 | 减半或换粗粮 |
| 粗粮 | 玉米、红薯、杂粮饭 | 优先选，饱腹感强 |
| 避免 | 炒饭、炒面、油条 | 油量大，热量翻倍 |

## 七、食堂场景判断规则

### 窗口选择优先级
1. **自选窗口**（称重计价）> 套餐窗口 > 小炒窗口 > 面食窗口
2. 自选窗口可以控制分量和搭配，性价比最高
3. 小炒窗口油量大，但蛋白质足，适合增肌

### 一餐搭配公式
- **减脂餐**：1素菜(绿叶) + 1优质蛋白(蒸/煮) + 半份粗粮主食
- **增肌餐**：1肉菜 + 1蛋/豆制品 + 1素菜 + 1份主食
- **省钱餐**：1豆腐/蛋 + 1素菜 + 1份米饭（总价<8元）
- **均衡餐**：1荤 + 2素 + 1主食（最推荐日常吃）
- **生理期餐**：1温热汤类 + 1红肉/补铁菜 + 1温性蔬菜 + 1份主食

## 八、运动与饮食联动

| 运动类型 | 运动后建议 | 食堂方案 |
|---------|----------|---------|
| 跑步/有氧 | 补充碳水+少量蛋白 | 香蕉+牛奶，或正常吃饭 |
| 力量训练 | 30分钟内补蛋白质 | 2个鸡蛋或1份鸡肉 |
| 无运动 | 正常饮食 | 控制碳水，增加蔬菜 |
| 高强度运动日 | 适当增加主食 | 正餐加一份粗粮 |

**原则**：运动量大→多吃点主食和蛋白质；不运动→减少主食，保持蛋白质和蔬菜。

## 九、情绪化进食应对策略

### 检测信号
| 信号 | 可能原因 | AI 应对 |
|------|---------|--------|
| 深夜连续记录高热量食物 | 压力/焦虑/无聊 | 温柔提示+低热量替代（热牛奶/水果） |
| 考试周/ddl前后外卖暴增 | 没时间/压力大 | 推荐食堂快又好的选择 |
| 连续3天吃甜食 | 情绪低落/经前综合征 | 推荐水果替代，关心情绪 |
| 连续吃同一窗口 | 懒得想/习惯 | 推荐相似口味的新窗口 |

### 应对原则
1. **先处理心情，再处理事情**——先关心用户情绪，不指责
2. **给替代方案，不只说"别吃"**——"少吃炸鸡"不如"试试食堂的少油版炸鸡腿"
3. **降低门槛**——推荐容易获得的健康替代，不要求用户跑很远
4. **允许偶尔放纵**——"今天吃了就吃了，明天咱们吃好点就行"

## 十、身体状况×饮食忌口速查

> **核心原则**：用户的身体状况（永久/临时）直接影响推荐安全性和合理性。推荐前必须检查当前身体状况，主动避坑。

### 永久性身体状况（记入 base.json）

| 身体状况 | 忌口/限制 | 食堂替代方案 |
|---------|----------|-------------|
| 食物过敏（酒精/海鲜/花生等） | **绝对避开**过敏源 | 其他蛋白质来源替代 |
| 脾胃虚弱/慢性胃炎 | 忌生冷、忌辛辣、忌油腻 | 温热软烂食物（粥/汤/蒸菜），少粗粮多白饭 |
| 乳糖不耐受 | 忌牛奶 | 豆浆、酸奶 |
| 麸质不耐受 | 忌面食 | 米饭、米粉 |
| 痘肌/脂溢性皮肤 | 忌高糖、忌辛辣、忌油炸 | 清淡蒸煮菜、少甜食 |
| 胆囊问题 | 忌高脂 | 低油蒸煮菜，少蛋黄 |
| 高尿酸/痛风 | 忌海鲜、忌啤酒、忌内脏 | 鸡蛋/豆腐/蔬菜替代蛋白 |

### 临时性身体状况（记入 modes/{mode}/health.json）

| 临时状况 | 忌口/限制 | 食堂替代方案 | 恢复后 |
|---------|----------|-------------|--------|
| 嗓子疼/咽炎 | 忌辣、忌干硬、忌油炸 | 温热软食（粥/汤/蒸蛋/面条） | 症状消失后恢复 |
| 感冒/发烧 | 忌生冷、忌油腻、忌甜食 | 热粥/鸡汤/清淡蒸菜 | 退烧后逐步恢复 |
| 拉肚子/肠胃炎 | 忌油腻、忌辛辣、忌生冷、忌粗纤维 | 白粥+咸菜、蒸蛋、清汤面 | 好了后先吃软食2天 |
| 拔牙/口腔溃疡 | 忌硬、忌辣、忌酸、忌烫 | 凉粥/凉面/蒸蛋/豆腐 | 伤口愈合后恢复 |
| 扁桃体发炎 | 忌辣、忌油炸、忌坚硬 | 温热软食（粥/汤/蒸蛋） | 症状消失后恢复 |
| 术后/住院恢复期 | 遵医嘱！ | 医生允许的范围内推荐 | 出院后逐步恢复 |
| 便秘 | 多纤维、多水 | 多蔬菜（绿叶）、红薯 | 改善后正常 |
| 蚊虫叮咬/皮肤过敏 | 少辛辣刺激 | 清淡饮食 | 好了后正常 |

### 安全检查优先级

评估顺序：**过敏/忌口冲突 → 身体状况冲突 → 目标匹配度 → 结构建议**

如果检测到安全问题，立刻打断优先报出。

### AI 对身体状况的处理流程

1. **永久状况**：首次采集时记录，之后每餐推荐自动避开
2. **临时状况**：用户随口提及即触发，好了就清除
3. **检测提醒**：连续2天记录与身体状况不符的食物 → 温柔提醒
4. **医嘱优先**：用户说"医生说不能吃XX"→ 无条件遵从

## 十一、食物不耐受常见类型

| 不耐受类型 | 常见症状 | 食堂替代 |
|-----------|---------|---------|
| 乳糖不耐受（牛奶） | 腹胀/腹泻/放屁 | 豆浆、酸奶（乳糖少） |
| 麸质不耐受（面食） | 腹胀/疲劳/头痛 | 米饭、米粉 |
| 组胺不耐受（发酵食品） | 头痛/皮肤潮红/心悸 | 新鲜食物，少腌制品 |
| 海鲜过敏 | 皮疹/瘙痒/呼吸困难 | 鸡蛋/豆腐替代蛋白质 |
| 芒果过敏 | 嘴唇肿/瘙痒 | 其他水果替代 |

**发现机制**：用户记录"吃完XX不舒服"≥2次 → AI 主动提示 → 建议替代 → 记入 `intolerance.json`。

## 十二、口味偏好学习规则

当用户连续记录3天以上饮食后，AI 应总结：
1. **偏好菜品**：出现≥2次的菜品
2. **偏好窗口**：经常去的食堂窗口
3. **回避食物**：明确不喜欢的或忌口
4. **口味模式**：偏辣/偏甜/偏咸/偏清淡

基于以上，推荐时：
- 优先推荐偏好口味的新菜品（同窗口没吃过的）
- 每周推荐1-2个"没试过但大概率喜欢"的菜品
- 记录用户对推荐的反馈，调整下次推荐
